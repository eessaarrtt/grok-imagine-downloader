Некоторые изображения процентов 30-40 некорректно скачиваются, ну и ладно.

**Как установить:**

Откройте Chrome, зайди в chrome://extensions/, включи режим разработчика (переключатель справа вверху), нажми "Загрузить распакованное расширение" и выбери папку chrome_extension_grok_downloader. Всё, готово.

**Как использовать:**

Открой grok.com/imagine, в правом верхнем углу появится кнопка "Скачать изображения". Нажми на неё, откроется панель. Там можно настроить количество изображений (по умолчанию 100) и префикс имени файла (по умолчанию "coloring_page"). 

Есть три кнопки:
- Найти изображения - просто найдет все на странице
- Скачать все - скачает найденные
- Авто-скачивание - будет автоматически прокручивать страницу и скачивать до указанного количества

**Что умеет:**

Ищет все изображения на странице (включая base64), автоматически прокручивает страницу для загрузки новых, может скачать до 1000 штук, показывает прогресс, не скачивает дубликаты.

**Важно:**

Изображения скачиваются в папку загрузок. Работает только на grok.com/imagine. Для лучших результатов используй "Авто-скачивание". Некоторые изображения могут скачиваться некорректно (30-40%), это нормально.
